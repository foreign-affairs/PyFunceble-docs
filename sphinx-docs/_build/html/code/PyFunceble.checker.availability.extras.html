class PyFunceble.checker.availability.extras.subject_switch.SubjectSwitchRulesHandler(status: AvailabilityCheckerStatus | None = None)[source]

Bases: ExtraRuleHandlerBase

Provides our very own “subject switch” handler.

This handler will be used to detects a subject switch behavior from a server. In other words, this handler should be able to detect the following scenario and switch the original subject to INACTIVE.

  1. https://www.example.org/hello/world -> https://example.org/hello/world

  2. https://m.example.org/hello/world -> https://example.org/hello/world
